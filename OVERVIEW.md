# PI-Hole Overview

## What this project is
A clean, advanced Pi-hole setup for DNS filtering, privacy and home-lab learning.  
This file is the place to add your **screenshots**, **notes**, and **short explanations**.

## Why Pi-hole is useful
- Blocks ads on all devices
- Improves privacy
- Reduces tracking
- Helps learn DNS/DHCP basics
- Perfect Raspberry Pi project
